# Typescript Configuration

A shareable TypeScript configuration.

## Usage

Implement the configuration into `tsconfig.json`:

```json
{
    "extends": "@apst/tsconfig"
}
```

Or for library:

```json
{
    "extends": "@apst/tsconfig/lib"
}
```
